Montanaro World Languages
Includes:
- Big hero with subtle color background
- Interactive “Why Virtual” cards
- Tabbed course list (Mon–Thu only; 3 time slots each; ICS add-to-calendar buttons)
- Pricing + referral CTA; “More courses coming soon!”
- About page, Terms page scaffold, Self-placement quiz
- SASS structure in src/sass and compiled CSS in public/css
Tips:
- Put your logos and photos in assets/images using these names:
  logo-mark.png, hero-remote-beanbag.jpg, madrid-gran-via.jpg, barcelona-sagrada-familia.jpg,
  paris-arc-de-triomphe.jpg, learner-at-desk-sunset.jpg, child-raising-hand-tablet.jpg, two-learners-laptop.jpg
- (Optional) Place your PDF agreement at assets/docs/tutoring-agreement.pdf and update the Terms page link.
Run:
  npm run start   (serves locally)
  npm run dev     (watch SASS and rebuild CSS)
  npm run build   (minify CSS)
